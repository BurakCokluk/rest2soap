# Rest2SOAP Monorepo

This repository hosts multiple Go services and accompanying infrastructure assets for the Rest2SOAP platform.

## Layout

- `services/` houses independent Go microservices.
- `infrastructure/` contains Kubernetes manifests and platform tooling.
- `helm/` provides per-service Helm chart scaffolding.
- `.github/workflows/` defines CI pipelines for each service.
- `Makefile` offers helper targets for building services locally.
